S.O.M 現場運用 プライバシーポリシー

SHELTERONE CO., LTD（以下「当社」といいます）は、当社が提供するアプリ「S.O.M 現場運用」（以下「本アプリ」といいます）において取得する利用者情報を、以下の方針に基づき適切に取り扱います。

1. 取得する情報

本アプリでは、以下の情報を取得または取り扱う場合があります。

  • 氏名、メールアドレス、電話番号などのアカウント情報
  • ログインおよび認証に必要な情報
  • 避難所、拠点、タスク、チェック項目、進捗、コメント、通知確認などの業務データ
  • 写真、写真に付随するメモ、撮影日時、登録日時などの証跡情報
  • 避難所、拠点、タスク、物資運搬などに紐づく位置情報または経路情報
  • 端末情報、OS、アプリバージョン、操作ログ、不具合調査に必要なログ情報

2. 利用目的

取得した情報は、以下の目的で利用します。

  • 避難所開設および運営に関する業務支援のため
  • タスク管理、チェック記録、進捗共有、関係者間の情報共有のため
  • 写真による証跡の保存、確認、共有のため
  • 本人確認、認証、権限管理、セキュリティ確保のため
  • 不具合の調査、品質改善、利用状況の把握のため
  • お問い合わせへの対応のため

3. カメラおよび写真ライブラリの利用

本アプリは、タスク証跡写真の撮影または選択のために、端末のカメラおよび写真ライブラリへのアクセスを求める場合があります。取得した写真は、本アプリの業務記録機能の提供に必要な範囲で利用します。

4. 第三者提供

当社は、法令に基づく場合または利用者の同意がある場合を除き、取得した情報を第三者に提供しません。ただし、本アプリの提供、保守、運用、データ保存、認証、配信等に必要な範囲で、業務委託先またはクラウドサービス事業者に取り扱いを委託する場合があります。

5. 外部送信および保存

本アプリで入力または登録された情報は、本アプリの API サーバー、認証基盤、画像保存先、配信および運用基盤に送信・保存される場合があります。

6. 安全管理

当社は、取得した情報の漏えい、滅失、毀損、不正アクセス等を防止するため、必要かつ適切な安全管理措置を講じます。

7. 開示、訂正、削除等

利用者は、当社が保有する自身の情報について、開示、訂正、削除、利用停止等を求めることができます。手続きについては、下記お問い合わせ先よりご連絡ください。

8. 本ポリシーの変更

当社は、必要に応じて本ポリシーを変更することがあります。重要な変更がある場合は、適切な方法で通知または公表します。
